IsomFolio is a **local-first photo library manager** for macOS. It organises your photos into catalogs stored entirely on your machine — no cloud account, no subscription, no internet connection required.

## Who is it for?

- **Photographers** who shoot in volume and need fast culling (ratings, flags, keyboard shortcuts).
- **Privacy-conscious users** who want full control over where their photos are stored and processed.
- **Power users** who want AI assistance (face recognition) without surrendering data to a cloud service.

## What it does

| Capability | Details |
|---|---|
| **Catalog management** | Multiple named catalogs, each a self-contained directory |
| **Folder watching** | Add a folder; IsomFolio monitors it for new, changed, and deleted files automatically |
| **Thumbnails** | Generated in parallel, cached to disk, loaded instantly on subsequent launches |
| **Culling** | 1–5 star ratings, pick/reject flags, hide-rejects toggle |
| **Tagging** | Manual tags with autocomplete; AI-suggested tags that you accept or reject individually |
| **Smart albums** | Criteria-based (tags, ratings, flags, date range, file type) — auto-updating |
| **Manual albums** | Drag-and-drop to organise into collections |
| **Search** | Full-text and criteria search across all metadata |
| **People** | Face detection and clustering via an optional extension; group faces into named people |
| **Undo / Redo** | Full undo history for tag edits, ratings, and flags |
| **Extensions** | Install `.isfx` packages to add AI capabilities without modifying the core app |

## What it does not do

- **RAW processing or editing** — IsomFolio is a library manager, not an editor. Open photos in your editor of choice.
- **Cloud sync** — by design. All data stays local.
- **Export / print** — not yet. Use the "Show in Finder" context menu action to hand off to another tool.

## Supported platforms

| Platform | Status |
|---|---|
| macOS 12+ | Fully supported |
| Linux | Builds and runs; untested in production |
| Windows | Not currently supported |
