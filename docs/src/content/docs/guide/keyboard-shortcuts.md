## Navigation

| Shortcut | Action |
|---|---|
| `←` | Previous photo |
| `→` | Next photo |
| `↑` | Move up one row |
| `↓` | Move down one row |
| `Esc` | Cancel / back (close loupe, close modals) |
| `Cmd+A` | Select all |
| `Cmd+Shift+A` | Deselect all |
| `Cmd+R` | Sync the selected folder |

## View

| Shortcut | Action |
|---|---|
| `Space` | Toggle Loupe (single photo); with 2+ selected, open the review One-up; in the review, flip Survey ⇄ One-up |
| `E` | Toggle Preview mode |
| `I` | Toggle Info panel |
| `C` | Review selected photos in Survey (all at once, 2 or more) |
| `+` / `Cmd++` | Zoom in — larger grid thumbnails, or zoom the loupe image |
| `-` / `Cmd+-` | Zoom out — smaller grid thumbnails, or zoom the loupe image |
| `Z` | In the loupe: toggle 1:1 (actual pixels) / Fit |
| `\` | Toggle hide rejected photos |
| `?` | Toggle shortcut help |

## Culling

| Shortcut | Action |
|---|---|
| `P` | Flag as Pick |
| `X` | Flag as Reject |
| `U` | Flag as Unflagged |
| `R` | In the review: remove the focused frame from the comparison (whittle down) |
| `1` | Set rating: 1 star |
| `2` | Set rating: 2 stars |
| `3` | Set rating: 3 stars |
| `4` | Set rating: 4 stars |
| `5` | Set rating: 5 stars |
| `0` | Clear rating |

## Tagging

| Shortcut | Action |
|---|---|
| `T` | Focus the tag entry (opens the Info panel if needed) |
| `.` | Repeat last tag (apply to current selection) |
| `Cmd+Z` | Undo |
| `Cmd+Shift+Z` | Redo |

## Tips

**Culling a shoot efficiently**
Work in Loupe mode (`Space`) and use arrow keys + flag keys without taking your hands off the keyboard. Go through every photo, `X` the clear rejects, `P` the standouts. Then press `\` to hide rejects and work through the remaining photos with ratings.

**Batch ratings**
Return to the grid, `Cmd+A` to select all, then press `3` to give everything a baseline of 3 stars. Then go back through and upgrade your favourites to 4 or 5.

**Tag-repeat workflow**
When tagging a set of similar photos (e.g. all from the same location), type the tag in the Info panel for the first photo. Then use `→` to move to the next and press `.` to repeat — no need to re-type.
